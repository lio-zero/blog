# Code Golf

> [Code Golf](https://en.wikipedia.org/wiki/Code_golf) 是一种娱乐性的计算机编程比赛，参赛者努力获得解决特定问题的尽可能短的源代码。

以下是偶然间发现 [aem1k](https://aem1k.com/) 上的其中一个示例：

