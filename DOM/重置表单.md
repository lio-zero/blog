# 重置表单

包含表单的网页上的一个常见需求是将其重置为原始状态。

有两种方式将表单重置为原始状态：

- 重置按钮
- `reset()` 方法

重置按钮是一个 `input` 元素 `type="reset"`：

```html
<input type="reset" />
```

你也可以使用 JavaScript 以编程方式重置表单。

你只需要引用表单元素：

```js
const form = document.querySelector('form')
```

然后你可以调用 `reset()` 方法：

```js
form.reset()
```

这将清除表单中的所有元素，将它们恢复到原始状态。
