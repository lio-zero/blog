# 垃圾回收

> wiki：[垃圾回收](<https://en.wikipedia.org/wiki/Garbage_collection_(computer_science)>)（Garbage Collection，简称 GC）是一种自动内存管理形式。垃圾收集器尝试回收程序分配但不再引用的内存，这样的内存被称为垃圾。
