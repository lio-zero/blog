# Repo 托管服务

有不同的存储库托管服务平台，最流行的有：[GitHub](https://github.com/)、[GitLab](https://gitlab.com/) 和 [Bitbucket](https://bitbucket.com/)。

它们使用 Git 进行软件开发和版本控制的互联网托管提供商。它提供了 Git 的分布式版本控制和源代码管理功能，以及它自己的特性。
